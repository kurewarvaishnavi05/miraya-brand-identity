# MIRAYA by Garima - Brand Identity Rules

All AI agents working on this project MUST strictly adhere to the following brand identity rules:

- **Brand Name**: MIRAYA by Garima
- **Luxury Aesthetic**: Every page must look like a premium designer fashion brand similar to Sabyasachi or Dior. Use a clean, elegant layout with cinematic lighting and high-end fashion photography style.
- **Color Palette**: Maintain the same luxury color palette throughout the entire website: Black (`#111111`), Burgundy (`#5B152A`), Gold (`#D4AF37`), and White/Off-White (`#F8F5F0`).
- **Logo Usage**: Use the provided MIRAYA logo consistently in the Navbar, Footer, Login page, Admin Panel, Favicon, and any loading screens.
- **Custom Imagery**: 
  - NO generic stock images.
  - Hero banners, category banners, and promotional sections must use custom AI-generated fashion images matching the category.
  - Every image should feature premium Indian fashion models wearing outfits from the respective category (Lehenga, Saree, Kurti, Anarkali, etc.).
- **Consistency**: Keep all images, typography, and styling consistent in quality, colors, and branding across both the storefront and the admin dashboard.
- **Responsiveness**: Ensure the entire luxury experience is fully responsive and flawlessly translates to mobile devices.
